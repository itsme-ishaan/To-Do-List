TaskFlow ek powerful aur aesthetically pleasing task management application hai. Ise **Glassmorphism UI** principles par design kiya gaya hai, jo ise ek futuristic aur clean look deta hai. 

---

## 🎯 Key Features

### ✨ Modern UI/UX
* **Frosted Glass Effect:** Modern glassmorphism design with backdrop filters.
* **Responsive Layout:** Laptop, tablet, aur mobile par flawlessly kaam karta hai.
* **Animations:** Smooth transitions aur interactive toast notifications.

### 📝 Task Operations
* **CRUD Functionality:** Tasks ko create, read, update, aur delete karne ki full flexibility.
* **Smart Categorization:** Priority levels (High, Medium, Low) set karein.
* **Progress Tracking:** Real-time statistics ke saath apne completed aur pending tasks dekhein.
* **Deadlines:** Due dates add karein taaki koi kaam miss na ho.

### 🔐 Security
* **User Auth:** Secure registration aur login system.
* **Session Management:** Protected routes taaki sirf authorized users hi apna data dekh sakein.
